Atualizado:2000-01-02 00:00:00.0

Descrição: Linhas de Projeto de Ordem de Serviço (receita planejada) versus Desembolsos de Projeto (custos)

Comentário/Ajuda: O Relatório de Margem de Projeto para Ordens de Serviço compara as Linhas de Projeto (receita planejada) contra os Desembolsos de Projeto (esforço real e custos). O valor da margem é calculado subtraindo os custos (desembolso, issue) da receita (linha). Os custos estão na moeda do Esquema Contábil - as linhas estão na moeda da lista de preço do projeto.
