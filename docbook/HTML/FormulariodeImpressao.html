Comentário/Ajuda: Define os documentos que você usa para esta Empresa/Organização. Note que o formato do cheque é definido na Janela "Banco (Conta)".< p> O Formato de Impressão que você define em um Tipo de Documento tem a maior prioridade (ex. O padrão é que o conjunto de Formatos de Impressão seja definido para todas as organizações da Empresa (Organization=*).
