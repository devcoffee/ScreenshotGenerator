Descrição: Converter senhas existentes em hashes

Comentário/Ajuda: This process will overwrite existing user passwords with a salted SHA-512 hash of the password so that they cannot be recovered if your database is compromised. (Note: If your password column is currently encrypted, the hash will also be encrypted.)
