Processo: Fluxo de Trabalho para Empresa

Criado:2004-09-28 11:18:29.0

Atualizado:2000-01-02 00:00:00.0

Descrição: Move o fluxo de trabalho personalizado para a empresa atual

Comentário/Ajuda: A arquitetura de segurança evita que você adicione personalizações/modificações específicas de empresas aos fluxos de trabalho padronizados existentes. Este processo move a sua personalização feita no nível do sistema para esta empresa para adicionar detalhes específicos desta empresa.

Table 1.1. FluxodeTrabalhoparaEmpresa Parâmetros
Nome Nome da coluna Referência Valores(Padrão) Descrição Comentário/Ajuda
Fluxo de Trabalho AD_Workflow_ID Tabela Direta Fluxo de Trabalho ou combinação de tarefas O campo "Fluxo de Trabalho" identifica um Fluxo de Trabalho único no sistema.
Empresa AD_Client_ID Tabela Direta Empresa/Locatário para esta instalação. Uma Empresa é uma Companhia ou uma Entidade Legal (pessoa jurídica). Dados não podem ser compartilhados entre Empresas. Locatário é um sinônimo para Empresa (Client).
