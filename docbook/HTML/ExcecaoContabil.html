2. Tabela: Linha - GL_DistributionLine

[DADOS DE EXEMPLO]

Descrição: Linha de Alvo de Distribuição de Razão Geral

Comentário/Ajuda: Se os critérios de combinação de contas da distribuição forem atingidos, a contabilização na combinação de contas é substituída pelas combinações de contas das linhas de distribuição. A distribuição é pro-rateada com base no valor percentual das linhas. Um valor percentual de 0 (nulo) indica que o resto será 100. Só pode haver uma única linha com 0 porcento.
