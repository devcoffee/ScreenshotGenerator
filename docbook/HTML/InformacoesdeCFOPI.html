Organização Tabela Direta c.AD_Org_ID Operador:= Função:null Entidade organizacional dentro da Empresa Uma "Organização" é uma unidade de sua "Empresa" ou "Entidade Legal" - os exemplos são loja, departamento. Você pode compartilhar dados entre organizações.
Chave de Busca Texto Curto (String) c.Value Operador:Like Função:Upper Chave de pesquisa para o registro no formato necessário - tem que ser único e exclusivo Uma "Chave de Busca" fornece a você um método rápido de encontrar um registro em particular. Se você deixar a chave de busca vazia, o sistema automaticamente cria um valor numérico. A seqüência de documento usada por este número de recuperação é definida na janela "Gerenciamento de Seqüência" com o nome "DocumentNo_< TableName> ", onde TableName é o nome real da tabela (ex: C_Order).
