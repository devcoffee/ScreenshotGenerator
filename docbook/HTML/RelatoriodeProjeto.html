Descrição: Gerenciamento de Ciclos de Relatórios de Projeto

Comentário/Ajuda: A Janela "Ciclo de Projeto" define os passos associados a um Projeto específico. Você pode querer definir diversos Ciclos de Projeto para diferenciar os diferentes tipos de projeto. Os Ciclos de Projeto podem usar todos ou um subconjunto dos Estados de Projeto (ex. Seu ciclo de projeto de oportunidade pode incluir os passos desde a prospecção até o contrato - o ciclo de serviços pode incluir passos desde a instalação até o aceite do cliente).
