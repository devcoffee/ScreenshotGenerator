Descrição: Gerenciamento dos Elementos de Contas

Comentário/Ajuda: A Janela "Elemento Contábil" é usada para definir e manter o Elemento Contábil e os Elementos Definidos pelo Usuário. Um dos segmentos de contabilidade é o seu segmento de contabilidade natural (Plano de Contas). Você pode acrescentar um novo Elemento Contábil para relatórios paralelos ou para segmentos de contabilidade definidos pelo usuário.
