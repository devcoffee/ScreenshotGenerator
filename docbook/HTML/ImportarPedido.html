Comentário/Ajuda: Por favor, faça as seleções Y/N (Sim/Não) no Formato do Carregador de Importação explicitamente (ex. como constantes) e não as deixe NULL (NULAS). Um valor NULL (NULO) significa "não conhecido". Em geral, caixas de seleção não são selecionadas se o valor não for "Y" (i.e. poderia ser nulo ou qualquer outro valor). Todos os testes são feitos com "Y" ou "not Y" - i.e. valores NULL não são "not Y", mas NULO (não conhecido) - veja as instruções da Linguagem SQL.
