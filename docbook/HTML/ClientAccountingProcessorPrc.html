Descrição: Client Accounting Processor

Comentário/Ajuda: The client accounting processor allows a user to run the accounting processor in a client session. The system configurator parameter CLIENT_ACCOUNTING must be enabled (configured as [I]mmediate or [Q]ueue).

Table 1.1. ClientAccountingProcessor Parâmetros
Nome Nome da coluna Referência Valores(Padrão) Descrição Comentário/Ajuda
Esquema Contábil C_AcctSchema_ID Tabela Direta Regras para contabilidade Um "Esquema Contábil" define as regras usadas na contabilidade tais como método de custos, moeda corrente e calendário.
Tabela AD_Table_ID Tabela Direta Informação da Tabela no Banco de Dados A Tabela no Banco de Dados fornece informações das Definições da Tabela.
