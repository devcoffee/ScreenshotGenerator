Descrição: Gerenciamento dos Papéis de Impressão

Comentário/Ajuda: Tamanho, Orientação e Margens do Papel da Impressora. O Código de Validação contém o nome do Tamanho da Mídia. The names of the media sizes correspond to those in the IPP 1.1 RFC 2911 - http://www.ietf.org/rfc/rfc2911.txt If the name is not found, the SizeX/Y with the dimension is used. Size and dimension are ignored, if a valid media size name is used.
