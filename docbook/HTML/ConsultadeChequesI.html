Um registro desativado não está disponível para seleção, mas está disponível para relatórios. Existem duas razões para desativar um registro ao invés de excluí-lo: (1) O sistema exige o registro para fins de auditoria. (2) O registro é referenciado por outros registros. Ex. você não pode excluir um Parceiro de Negócios, se existirem faturas para este parceiro de negócios. Você desativa o Parceiro de Negócios e previne que este registro seja usado para entradas futuras.
