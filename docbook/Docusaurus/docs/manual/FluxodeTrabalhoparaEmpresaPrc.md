## Processo: Fluxo de Trabalho para Empresa

</div>

</div>

</div>

<span class="emphasis"> *Criado:*</span>2004-09-28 11:18:29.0

<span class="emphasis">*Atualizado:*</span>2000-01-02 00:00:00.0

<span class="emphasis"> *Descrição:* </span>Move o fluxo de trabalho
personalizado para a empresa atual

<span class="emphasis"> *Comentário/Ajuda:* </span>A arquitetura de
segurança evita que você adicione personalizações/modificações
específicas de empresas aos fluxos de trabalho padronizados existentes.
Este processo move a sua personalização feita no nível do sistema para
esta empresa para adicionar detalhes específicos desta empresa.
